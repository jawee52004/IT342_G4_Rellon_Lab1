# IT342 Mini App – User Registration & Authentication

## Description
This project demonstrates a basic user registration, login, dashboard access, and logout flow.

## Technologies
- Backend: Spring Boot 
- Web: React 
- Mobile: Android 

## Project Structure
- backend/ – Spring Boot backend
- web/ – React frontend
- mobile/ – Android app 
- docs/ – System documentation and diagrams


How to run
1. Run Backend "run Backend Application"
2. Run Frontend "npm start"
